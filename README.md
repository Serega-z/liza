# Платформа для обучения добровольцев Лиза Алерт (статичная вёрстка)
В рамках программы обучения в "Яндекс Практикум" реализована вёрстка страниц **"Курсы"** и **"Профиль"** с информацией о курсах подготовки и участниках спасательных отрядов.
------ 
## Курсы: 
* на странице находятся карточки с краткой информацией о курсах (категория, описание, количество часов и занятий, статус); 
* соответствующие фильтры.

## Профиль: 
* содержит информацию о пользователе, поля для ввода и редактирования личных данных.


Используемые технологии:
* **HTML**
* **CSS**

Файлы структурированы по правилам *БЭМ Nested*.
